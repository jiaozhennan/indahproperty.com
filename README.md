# Pacific Garden Puri - 巴厘岛豪华房产项目网站

这是一个为巴厘岛豪华房产项目 Pacific Garden Puri 设计的现代化响应式网站。网站展示了项目的各种房产类型、特色优势、新闻资讯和联系方式，为潜在客户提供全面的项目信息和视觉体验。

## 技术栈

- **前端框架**: React 18+ (使用函数组件和Hooks)
- **编程语言**: TypeScript
- **构建工具**: Vite
- **路由**: React Router
- **样式**: Tailwind CSS v3
- **动画**: Framer Motion
- **图标**: Font Awesome
- **表单验证**: Zod
- **状态管理**: Context API
- **通知**: Sonner
- **部署**: GitHub Pages

## 项目结构

项目采用模块化结构，将不同功能拆分为独立的组件和页面，便于维护和扩展：

```
src/
├── components/         # 可复用的UI组件
├── contexts/           # React Context API上下文
├── hooks/              # 自定义Hooks
├── lib/                # 工具函数
├── locales/            # 多语言翻译文件
├── pages/              # 页面组件
├── App.tsx             # 应用入口组件
├── main.tsx            # 应用渲染入口
└── index.css           # 全局样式
```

## 核心功能

1. **多语言支持**: 支持中文、英文和印尼文三种语言切换
2. **响应式设计**: 适配桌面端、平板和移动设备
3. **动态导航栏**: 滚动时自动调整样式，支持移动端汉堡菜单
4. **房产展示**: 展示多种类型的豪华房产，包含图片、价格和详细信息
5. **新闻资讯**: 提供最新的房产市场动态和项目更新
6. **联系表单**: 用户可以提交咨询信息，数据存储在浏览器localStorage中
7. **动画效果**: 使用Framer Motion为页面添加平滑的动画和过渡效果

## 性能优化

1. **组件拆分**: 将大型组件拆分为小型可复用组件，提高代码可维护性
2. **React.memo**: 使用React.memo减少不必要的组件重渲染
3. **图片懒加载**: 实现图片懒加载，提高页面加载速度
4. **防抖和节流**: 对滚动和点击事件应用防抖和节流优化
5. **代码分割**: 优化代码结构，提高首次加载性能
6. **字体优化**: 使用字体预加载和font-display: swap提高字体加载体验

## 推荐的后台解决方案

由于当前项目是纯前端实现，所有表单数据存储在localStorage中。对于实际的生产环境，建议使用以下后台解决方案：

1. **Node.js + Express**: 构建RESTful API服务
2. **数据库**: MongoDB或MySQL存储房产信息和用户咨询
3. **认证**: JWT实现用户认证和权限控制
4. **文件存储**: AWS S3或阿里云OSS存储房产图片
5. **内容管理系统**: 可考虑集成Strapi或WordPress作为CMS

## 开发说明

1. 安装依赖:
```bash
pnpm install
```

2. 启动开发服务器:
```bash
pnpm dev
```

3. 构建项目:
```bash
pnpm build
```

4. 部署到GitHub Pages:
```bash
pnpm deploy
```
